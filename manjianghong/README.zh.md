# 《满江红·怒发冲冠》作者归属 —— 计量语言学考辨

用现代计量文体学（stylometry）方法，比较待考之《满江红·怒发冲冠》与 ① 岳飞确信作品、
② 明代征贺兰山名将**王越**的诗（"伪托说"指认的候选作者圈）、③ 外部对照，给出**数据**，
据数据作克制判断。

> ⚠️ **核心声明**：① 代码（`compute_metrics.py`）**只计算数据、不写结论**——输出是 `数据/` 下
> 一批可复核的 CSV；解释与判断由人去读数据（见下文第五节，明确标注为"对数据的解读"）。
> ② 岳飞可靠传世作品仍很少，且体裁不齐（词/诗/文混杂），小样本使所有数值**统计功效有限**；
> 本研究下文用**留一法正确率**和**组内基线**把"这套方法到底有多可信"也量化出来。

## 三、代码做了什么（只产出数据）

`compute_metrics.py` 计算并写出（全部纯标准库实现，算法见源码注释）：

| 输出 CSV | 内容 |
|---|---|

| `05a/05b_距离矩阵` | 全 40×40 篇距离矩阵（二元组余弦 / 单字 JS 散度），供聚类 |
| `06_最近质心分类_LOO判别.csv` | 最近质心分类；**留一法(LOO)正确率**=这套方法能否在"已知"诗上区分岳飞/王越；待考判给谁 |
| `06b_LOO逐篇判别明细.csv` | 每一篇已知诗的留一判别结果 |
| `07_组内基线_vs_待考.csv` | 组内两两相似度分布 vs 待考到组的相似度（**z 分**=待考是否落在该作者的"自身离散度"之内） |
| `08_特征计数_地理敌称母题.csv` | 逐篇：地理指向/敌称/母题词元**计数**（只计数，异常与否留给人判断） |

## 五、对数据的解读（人工，非代码输出）

**结果随"特征视图"翻转，且方法自评可信度不高——这本身是最重要的发现。**

1. **抽象文体（虚词 / Burrows Delta）→ 偏王越**：虚词余弦 王越 0.299 > 苏轼 0.185 > 岳飞 0.171；
   Burrows Δ 王越 0.496 < 岳飞 0.752（越小越近）。**但** `06` 显示：虚词视图的 LOO 正确率仅 0.64，
   且**只认对 4/13 首岳飞自己的诗**（9 首被误判成王越）——王越 23 首同质七律形成"吸引子"，
   把一切都吸过去。所以这一路的"判给王越"**不可信**。

2. **内容词搭配（二元组）/ 语言模型 / 组内基线 → 偏岳飞**：
   - 二元组 containment：岳飞 0.0435 ≈ 4× 王越 0.0109；`02` 中除族谱近重本外，待考的最近邻几乎都是岳飞诗（小重山、过张溪、题翠岩…）。
   - 等长截断后的字符二元组语言模型困惑度：**岳飞 950 < 王越 990 < 苏轼 999**（岳飞略优；但 PPL 逼近词表 999，模型近乎均匀，差异属弱信号）。
   - 组内基线 `07`：待考落在**岳飞单字/二元组云团之内**（z=+0.42 / +0.26，比岳飞诸作彼此还略近），却落在**王越云团之外**（各视图 z<0）。
   - 二元组视图 LOO（最能认对岳飞、11/13）把待考**判给岳飞**。

3. **"用语时代错置"旧诘难被数据排除**：`08` 显示泛化敌称（胡虏/匈奴/夷狄/虏）正是岳飞惯用
   （五岳祠盟记一篇就 9 次；送紫岩用可汗/阏氏；题翠岩用金酋）。母题上，待考一篇**集齐全部 6 类**
   （功名/白发/收复/魏阙/靖康/饮血）岳飞反复出现的母题——与岳飞**高度相容**，但这也正是**伪托者刻意拼合**会有的特征。

4. **唯一稳健、可解释的异常仍是地理**：`08` 中待考 `geo_西北_贺兰=2、geo_东北中原=0`，
   是**全部岳飞系作品里独一份**——岳飞凡含地名者（登黄鹤楼 5、送紫岩 3、五岳祠盟 4、题骤马冈 1）
   **无一例外指向东北—中原**（其抗金战场）。族谱"原作"《与祝允哲述怀》此处恰作"金城"（方位正确），
   "贺兰山"实出自祝允哲和词。这正是夏承焘据以指认明将王越（征贺兰山）幕府的内证基础。

## 六、结论

> **计量证据无法判定作者**：换一个特征视图，结论就在"岳飞"与"王越"间翻转，而方法在已知诗上的
> 留一正确率只有 0.64–0.81、尤其**认不准岳飞本人的诗**——所以任何单一"判给"都不足为凭。
>
> 在信号确实出现处：**内容词搭配、语言模型拟合、组内云团位置、母题/敌称习惯，均倾向"与岳飞相容"**；
> 偏向王越的只是抽象虚词风格，而那条路的分类器已被证明认不准岳飞，其"王越票"很弱。
>
> 因此最稳妥的表述是：**就语言痕迹而言，《怒发冲冠》与岳飞高度相容、计量上无法证伪，
> 也无法把它判给王越；但"相容≠证真"。计量语言学能排除"敌称/用语时代错置"一类旧诘难，
> 却消解不了两大非风格疑点——①『贺兰山』的西北地理指向（全岳飞系独一份）；
> ② 宋元传世链的缺环。二者属史源学/内证范畴，目前仍存疑而无定论。**

## 七、局限

- 样本极小（岳飞 848 字、其中词仅 2 首）→ 文体统计置信度低，`06`/`07` 已把这一点量化。
- **体裁混淆**：待考是词，岳飞语料以诗文为主、王越全为诗 → 余弦/二元组差异部分来自体裁而非作者。
- 字符二元组 LM 在此数据量下近乎均匀（PPL≈词表大小），其排序仅供参考。
- 地理/敌称/母题词典是人工设定的特征工程；`08` 只给计数，判断留给读者。
